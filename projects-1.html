Ovaflow

Ovaflow is a light-weight rhythm game fueled by the user community. (similar to step-mania) Currently developed for the Android platform only. The objective of the game is to press the correct sequence of keys or buttons as notes cascade down the screen towards a special buttons on the screen. The game's score will be calculated based on the accuracy of the button presses atop some precision tolerance, and might end prematurely if the user makes too many errors. The differentiating factor is that all the "beatmaps" are created by users which may be uploaded onto the central repository and downloaded along with avatars.

What started as a simple prototype for a class ended as a fully functional game. I was primarily responsible for the Android platform but also built up the initial framework for the server

The project is divided into two parts, the Android platform (Client) and the server repository.

Client

The Client only supports Android devices. The rendering is done by using OpenGL ES 2.0. The rendering is displayed via a SurfaceView. The downloading of songs is done by sending REST-ful requests to the server and using AsyncTasks to fetch the songs over a stream.

Server

The server provides the user high scores, avatars, and a song repository accessed via JAX-RS
